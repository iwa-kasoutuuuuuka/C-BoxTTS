# C-Box TTS 日本語ポータブル版 仕様書

本ソフトウェアは、Python環境を含まないWindows環境でも動作するように設計された自己構築型ポータブルアプリケーションです。

## フォルダ構成
```text
C-BoxTTS/
├── C-BoxTTS.exe           # メイン実行ファイル
├── python_embeded/        # 同梱されたPython本体（初回構築の土台）
├── packages/              # [手動作成用] .whlファイルの配置場所（オフライン用キャッシュ）
├── README.md              # 簡易説明書
└── ポータブル版仕様書.md  # 本仕様書
```

## セットアップ仕様
起動時に必須ライブラリ（PyTorch等）の存在を確認し、不足している場合は「自動セットアップ」画面を表示し、自動で `python_embeded/Lib/site-packages` 内に必要なパッケージをインストールします。

### 高速・オフラインセットアップ手順
ネットワークが遅い、またはオフライン環境で構築したい場合は、以下のファイルを `C-BoxTTS.exe` と同じ場所に配置してから起動してください。

1. **packages フォルダの作成**: 
   - `packages` フォルダを作成し、その中に [PyTorch CUDA版のWheel](https://download.pytorch.org/whl/cu124/torch-2.6.0%2Bcu124-cp310-cp310-win_amd64.whl) などの `.whl` ファイルを配置すると、ダウンロードをスキップして高速にローカルインストールします。

## 主要機能と最適化の更新
- **推論高速化（GPU向けAMP）**: GPU（CUDA）上での演算に自動混合精度（FP16/AMP）を適用。VRAM消費を抑制しつつ、テンソルコアを活用して推論速度を約12%〜20%向上させました。
- **システム保護（CPUスレッド制限）**: CPU推論時のスレッド競合によるPC全体のハング・高負荷を避けるため、内部の論理スレッド数を上限 `4` に制限しました。
- **文字数カウント**: 入力エリア下にリアルタイムの文字数を表示。長文生成の目安になります。
- **多言語対応拡大**: Multilingualモデルにおいて、日本語(ja), 英語(en), 中国語(zh), 韓国語(ko)を含む主要23言語の選択に対応しました。
- **詳細設定の拡充**: 感情の強さ、抑揚(CFG), 多様性(Temp)に加え、話速(Speed)の調整をメイン画面から直感的に行えるようになりました。

## 技術的特記事項
- **エンコーディング**: 標準入出力を UTF-8 に固定し、Windows環境特有の絵文字出力等によるクラッシュを防止。
- **二重起動防止**: `multiprocessing.freeze_support()` によるプロセスの整合性確保。
- **容量極小化**: ビルド時に PyTorch や transformers などの大容量ライブラリ（数GB）を除外し、かつ PaddlePaddle や Matplotlib などの不要モジュールも完全に排除したことで、配布サイズを約 **70MB** までスリム化しました。
- **セットアップの堅牢化**: 実績のある `setuptools==75.0.0` を固定導入し、依存パッケージの自動インポート時のクラッシュ（pkg_resources不足）を未然に防止。
- **話速調整**: `librosa.effects.time_stretch` を使用した高品質なタイムストレッチ処理。

## 堅牢性およびセキュリティ向上（2026-06-16 更新）
- **再生ファイルロック競合の回避（PermissionError防止）**: 音声プレビュー再生時、pygameが以前に再生したファイルを掴むことで発生する `PermissionError` を防ぐため、書き込み・再生前に明示的に `stop()` および `unload()` を実行する安全設計を導入。
- **音声結合における防御的ガード**: テキスト分割結果や合成結果に空の値（`None` や空テンソル、非テンソル）が混入した場合でも、エンジンがクラッシュせず適切に除外して結合処理を続行する例外ガードを `merge_audio` に追加。
- **入力サニタイズと早期エラーハンドリング**: テキストエリアが空、または空白文字だけの場合の合成要求に対して、早期に警告を出し、無駄な推論リソースの消費や内部エラーを防ぐ処理を強化。

## 注意事項
- 初回のライブラリ自動インストール完了時には、ディスク容量が約 3GB〜5GB 必要です。
- GPU（CUDA）を利用する場合は、最新のNVIDIAドライバがインストールされている必要があります。
- 低スペックPCでは、モデルのロードや音声生成の初回開始に時間がかかる場合があります。
